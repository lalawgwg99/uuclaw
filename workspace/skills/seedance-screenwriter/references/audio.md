# 音频技巧

## 格式规范

- 清晰地用单独的句子陈述音频需求
- 示例："音频包含柔和的雨声和远处的雷声。男旁白说：'欢迎来到未来。'"

---

## 对话（防字幕）

❌ 失败：`女人说："台词"`
✅ 有效：`女人说：'台词'`

### 对话类型

| 类型 | 示例 |
|-----|------|
| 直接对话 | 老师说：'今天我们将探索光合作用' |
| 会话式 | 女人问：'我们去哪？' 男人答：'...' |
| 情感对话 | 若有所思地停顿，然后坚定地说：'...' 声音带着决心 |

### 对话长度

- **最佳**：8秒左右，自然朗读
- **太长**：15秒+，导致说话匆忙
- **太短**：单词如 "Hello"，导致沉默或乱码

### 多角色对话

```
穿红色连衣裙的女人问：'我们在哪里吃午饭？' 穿蓝色衬衫的男人回答：'市中心那家新意大利餐厅怎么样？'
```

### 发音修正

```
原始：Read on to get Fofur and Shridar's guidance
修正：Read on to get foh-fur's and Shreedar's guidance
```

## 字幕防止技巧

### 方法1 - 冒号格式（推荐）
```
✅ 侦探直视镜头说：'这里有些不对劲。'他的声音带着怀疑和决心。
```

### 方法2 - 明确否定
```
(no subtitles) 或 no subtitles, no text overlays
```

### 方法3 - 多次否定（顽固情况）
```
No subtitles. No subtitles! No on-screen text whatsoever.
```

## 环境音效

### 城市
```
音频：远处交通嗡嗡声，偶尔喇叭声，人行道脚步声，低沉交谈声，城市环境音带有微妙建筑回声
```

### 自然
```
音频：穿过树林的微风，鸟鸣声，树叶沙沙，远处流水声，宁静森林氛围
```

### 专业
```
音频：键盘打字声，空调嗡嗡声，低沉电话交谈声，纸张沙沙声，专业办公室氛围
```

### 烹饪
```
音频：煎锅嘶嘶声，切菜声，水沸腾声，餐具叮当声，厨房烹饪氛围
```

### 车间/技术
```
音频：工具叮当声，机器嗡嗡声，金属碰撞声，设备运转声，工业氛围
```

### 健身房
```
音频：哑铃叮当声，活力音乐，设备声音，呼吸声，激励能量
```

### 森林
```
音频：鸟鸣声，树叶沙沙声，风声，天然森林氛围，远处野生动物声音
```

### 海洋
```
音频：海浪声，海鸥叫声，风声，水流动声，海岸氛围
```

### 室内
```
音频：环境房间音，时钟滴答声，空调嗡嗡声，天然室内声学
```

## 音乐

| 类型 | 示例 |
|-----|------|
| 情绪音乐 | 振奋人心管弦乐，带有弦乐和铜管，逐步达到高潮，传达希望和决心 |
| 类型配乐 | 柔和爵士钢琴搭配细腻低音线，营造精致亲密氛围 |

## 音频幻觉预防

❌ 问题：不想要的"现场观众"笑声
✅ 解决：指定具体环境音频
```
差：角色讲笑话
好：角色讲笑话。音频：安静的办公室氛围，无观众声音，专业氛围。
```

## 高级序列

### 先...然后...技巧

**情感进展**：
```
角色开始困惑和不确定，然后逐渐变得自信和坚定，最后以满足的成就笑容结束。
```

**动作序列**：
```
她先在门口犹豫，然后深深吸一口气，最后坚决地推开门。
```
